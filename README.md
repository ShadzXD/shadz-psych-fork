Psych Engine 1.0.4 Fork


compile guide
Install the source code.
Open a terminal in the source code folder and run the command: haxelib install hmm
After hmm is installed, run: hmm install
If you are prompted to choose a location for the libraries, run: .haxelib
If everything has been installed correctly, use the command lime test cpp.
If no issues are found, the engine should compile without any errors.
